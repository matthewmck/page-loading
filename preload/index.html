HTML Preload Attribute

Preload will fetch resources on current page. It will set the resource to high priority and load it before standard resources. Because the browser couples resource fetching and execution of similar items, using the as attribute is recommended for preload.

As Options

  • script
  • style
  • audio
  • video
  • image
  • font
  • document
  • track
  • worker
  • fetch
  • object
  • embed
